Documentation sur les Directives Angular

Qu'est-ce qu'une Directive ?

Les directives Angular permettent de modifier le comportement des éléments du DOM. Il existe trois types principaux :

  • Directives Structurelles : Modifient la structure du DOM (*ngIf, *ngFor).
  • Directives d'Attribut : Changent l'apparence ou le comportement d'un élément ([style], [class]).
  • Directives Personnalisées : Créées par le développeur pour des besoins spécifiques.
