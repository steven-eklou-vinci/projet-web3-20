Documentation sur les Templates et le Data Binding

Qu'est-ce qu'un Template ?

Le template est la structure HTML qui définit l'interface utilisateur. Il peut inclure des balises HTML, des directives Angular et du Data Binding pour connecter la logique de l'application aux vues.

Qu'est-ce que le Data Binding ?

Le Data Binding est une fonctionnalité clé d'Angular permettant de synchroniser les données entre le modèle (classe TypeScript) et la vue (template HTML).

  • Interpolation :
  • Property Binding : Liaison unidirectionnelle vers les propriétés HTML.
  • Event Binding : Ecoute des événements utilisateur.
  • Two-Way Binding : Synchronisation bidirectionnelle des données.
