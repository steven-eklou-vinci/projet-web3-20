Documentation sur les Templates et le Data Binding

Qu'est-ce qu'un Template ?

Le template est la structure HTML qui définit l'interface utilisateur. Il peut inclure des balises HTML, des directives Angular et du Data Binding pour connecter la logique de l'application aux vues. Un template Angular est généralement associé à une classe TypeScript, et cette combinaison permet de définir une vue réactive.

Qu'est-ce que le Data Binding ?

Le Data Binding est une fonctionnalité clé d'Angular permettant de synchroniser les données entre le modèle (classe TypeScript) et la vue (template HTML). Voici les principaux types de Data Binding :

  • Interpolation : .
  • Property Binding : Liaison unidirectionnelle pour transmettre des valeurs aux propriétés d'un élément HTML (ex : [src], [disabled]).
  • Event Binding : Permet d'écouter et de réagir aux événements utilisateur comme (click) ou (input).
