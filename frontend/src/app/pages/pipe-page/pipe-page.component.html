Documentation sur les Pipes Angular

Qu'est-ce qu'un Pipe Angular ?

Un pipe dans Angular est une fonction qui permet de transformer des données dans les templates. Il peut être utilisé pour formater des dates, des monnaies, ou pour effectuer d'autres types de transformations de données. Un pipe reçoit une valeur en entrée et retourne la valeur transformée.

Structure d'un Pipe

Un pipe Angular est composé principalement de deux éléments :

  • Classe Pipe : Contient la logique de transformation des données.
  • PipeTransform Interface : Interface que la classe pipe implémente, avec la méthode transform().

Pour créer un nouveau pipe Angular, vous pouvez utiliser la commande suivante dans votre terminal : 

ng generate pipe nom-du-pipe

Utilisation d'un Pipe :

Pour utiliser un pipe dans un template, il faut d'abord importer le module qui contient le pipe, puis l'ajouter dans la section imports du composant. Ensuite, vous pouvez l'utiliser dans le template avec le symbole |.

Date sans pipe : {{ '2024-11-21' }}

Date avec pipe : {{ '2024-11-21' | customDate }}

Conclusion :

Les pipes permettent de simplifier la transformation de données dans les templates Angular, en rendant le code plus propre et lisible.
