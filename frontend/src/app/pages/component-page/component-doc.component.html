Documentation sur les Composants Angular

Qu'est-ce qu'un Composant Angular ?

Un composant Angular est une partie de l'interface utilisateur d'une application Angular. Chaque composant a une vue, une logique de comportement et un modèle de données. Un composant est défini par un template HTML, une classe TypeScript, et une feuille de style CSS/SCSS.

Structure d'un Composant

Un composant Angular est constitué de trois éléments principaux :

  • Template HTML : Le code HTML qui définit l'interface.
  • Classe TypeScript : Contient la logique de comportement, les données et les méthodes du composant.
  • Styles (CSS/SCSS) : Les styles visuels associés au composant.

Pour créer un nouveau composant Angular, vous pouvez utiliser la commande suivante dans votre terminal : 

ng generate component nom-du-composant
